# 🚀 TG2RUB

انتقال خودکار فایل از تلگرام به روبیکا و بله — سریع، ساده و پایدار

**پشتیبانی از مدیریت فایل‌های حجیم با تقسیم خودکار**

*** بیس این پروژه از caffeinexz/Tele2Rub میباشد و این نسخه بهینه شده آن هست.***

---

## 🧠 معرفی

**TG2RUB**

یک ابزار سبک و کاربردی است که فایل‌ها را از بات تلگرام دریافت کرده و به صورت خودکار به **روبیکا**، **بله** یا **هر دو** ارسال می‌کند.

کل فرایند به صورت **صف (Queue)** انجام می‌شود تا از بروز خطا و تداخل جلوگیری شود.

---

## ⚙️ نحوه کار

```text
Telegram Bot -> Queue -> Download/Prepare -> Send to Rubika / Bale / Both
```

* دریافت فایل از تلگرام
* ذخیره موقت در سرور
* ثبت در صف
* ارسال خودکار توسط **Worker**

## 🛠 نصب سریع (Auto Install)

نصب سریع پروژه:

```bash
bash <(curl -s https://raw.githubusercontent.com/DayiGorbay/TG2RUB/main/install.sh)
```

یا روش دستی (zip):

```bash
unzip TG2RUB.zip
cd TG2RUB
chmod +x install.sh
./install.sh
```

برای مدیریت سرویس:

```bash
sudo systemctl status tg2rub
sudo systemctl restart tg2rub
sudo journalctl -u tg2rub -f
```

این اسکریپت:
- اسکریپت نصب را مستقیم از GitHub دریافت و اجرا می‌کند
- پیش‌نیازها (`python3`, `python3-venv`, `python3-pip`) را نصب می‌کند
- محیط مجازی `venv` را می‌سازد و وابستگی‌ها را نصب می‌کند
- مقادیر `.env` را به‌صورت تعاملی دریافت و ذخیره می‌کند
- سرویس `systemd` با نام `tg2rub` را می‌سازد، فعال می‌کند و اجرا می‌کند
- فایل‌های پروژه را در مسیر ثابت `/opt/TG2RUB` نصب و نگهداری می‌کند

## ⚙️ تنظیمات

یک فایل `.env` در **ریشه پروژه** میباشد:

```env
API_ID=عدد_API
API_HASH=کد_API
BOT_TOKEN=توکن_ربات
ADMIN_TELEGRAM_ID=آیدی_عددی_مدیر_تلگرام
BALE_BOT_TOKEN=توکن_ربات_بله
BALE_ADMIN_CHAT_ID=آیدی_چت_مدیر_در_بله
RUBIKA_SESSION=rubsession
```

یا از فایل نمونه استفاده کنید:

```bash
cp .env.example .env
```

## 📌 دریافت API_ID و API_HASH از تلگرام

برای استفاده از پروژه، ابتدا باید API تلگرام دریافت کنید:

1. وارد سایت زیر شوید:
   👉 https://my.telegram.org

2. با شماره تلگرام خود وارد شوید

3. روی **API development tools** کلیک کنید

4. فرم را به شکل زیر پر کنید:

```text
App title: TG2RUB
Short name: T2R
```

5. پس از ثبت، مقادیر زیر به شما داده می‌شود:

* **API_ID**
* **API_HASH**

این مقادیر را در فایل `.env` قرار دهید.

درصورت مشکل در دریافت API ID و API HASH مقادیر در کانال تلگرام قرار گرفته
